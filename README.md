## 📌 Descrição do Projeto
Este projeto consiste em um aplicativo mobile desenvolvido em **React Native**, com o objetivo de criar um formulário de cadastro simples.

O aplicativo permite que o usuário insira informações pessoais e visualize os dados preenchidos na própria tela após o envio.


## 🚀 Funcionalidades
- Preenchimento de formulário com os campos:
  - Nome
  - Curso
  - Disciplina
  - Descrição
- Validação para garantir que todos os campos sejam preenchidos
- Exibição dos dados informados após o envio
- Limpeza automática dos campos após envio
- Interface estilizada com tema escuro
- Uso de hooks do React:
  - `useState`
  - `useEffect`


## 🧱 Tecnologias Utilizadas
- React Native  
- JavaScript  

### Componentes utilizados:
- SafeAreaView  
- View  
- Text  
- TextInput  
- Button  
- StyleSheet  


## 🎨 Interface
O aplicativo possui:
- Fundo em azul escuro (quase preto)
- Campos de entrada claros para melhor contraste
- Textos em branco
- Mensagem de sucesso destacada
